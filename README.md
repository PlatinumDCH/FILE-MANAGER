## Идея
Собрать функционал работы с папками и файлами в одном месте.Удобное управление по через терминал.

### Структура

```
├── main.py                     # Точка входа в приложение
├── core/
│   ├── __init__.py
│   ├── viewer.py               # Модуль для просмотра файлов и папок
│   ├── manipulator.py          # Модуль для создания, переименования, удаления
│   ├── mover.py                # Модуль для перемещения и копирования
│   ├── sorter.py               # Модуль для сортировки файлов
│   ├── searcher.py             # Модуль для поиска файлов
│   ├── metadata.py             # Модуль для изменения атрибутов и прав доступа
│   ├── size_checker.py         # Модуль для проверки и управления размером папок
│   └── archiver.py             # Модуль для работы с архивами
├── utils/
│   ├── __init__.py
│   ├── file_utils.py           # Вспомогательные функции для работы с файлами
│   ├── folder_utils.py         # Вспомогательные функции для работы с папками
│   ├── logger.py               # Журнал действий пользователя
├── config/
│   ├── settings.py             # Настройки проекта
│   ├── constants.py            # Константы
└── tests/
    ├── test_viewer.py
    ├── test_manipulator.py
    ├── test_mover.py
    ├── test_sorter.py
    ├── test_searcher.py
    ├── test_metadata.py
    ├── test_size_checker.py
    ├── test_archiver.py
    └── test_utils.py
```

## Функционал:
```
  1.Основные функции:
        - Просмотр файлов и папок
        - Создание, переименование и удаление файлов и папок
        - Перемещение и копирование файлов и папок
        - Сортировка файлов по расширению или другим критериям
        - Поиск файлов по имени или другим метаданным
        - Изменение свойств файлов (например, атрибуты, права доступа)
        - Проверка и управление размером папок
        - Работа с архивами (ZIP, RAR и т.д.)
        - Управление метаданными (даты создания, изменения и т.д.)
  2.Дополнительные функции:
        - Поддержка многопоточности для больших операций (например, копирование/перемещение множества файлов)
        - Ведение журнала действий пользователя
        - Визуальный интерфейс 
        - Интеграция с облачными сервисами (Google Drive, Dropbox и т.д.)
        - Поддержка командной строки
```
